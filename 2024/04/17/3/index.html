静态博客搭建总结报告

1. 框架选择

1.1 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

1.2 Github Pages

Github Pages 是一个静态站点托管服务，它直接从 Github 的仓库中获取静态页面文件，用来展示托管在 Github 上的项目。

1.3 选择原因

  • Hexo 框架简单，容易上手，并且有丰富的主题可供选择。
  • Github Pages 免费，并且可以绑定自己的域名。

1.4 缺点

  • 国内访问速度较慢。
  • 无法自定义域名。

1.5 优点

  • 免费，并且可以绑定自己的域名。
  • 国内访问速度较快。
